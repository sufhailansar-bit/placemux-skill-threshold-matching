PlaceMux Skill Threshold Matching

Project Overview

This project demonstrates a skill-threshold-based job matching system for the PlaceMux platform.

The system compares student skill scores against job requirements and generates match scores based on threshold validation.

Features

- Skill Threshold Validation
- Match Vector Generation
- Candidate Ranking
- Eligibility Checking
- Match Score Calculation

Example

Job Requirements

Skill| Required Score
Python| 70
Machine Learning| 65
Data Analysis| 60

Student Skills

Skill| Score
Python| 80
Machine Learning| 70
Data Analysis| 55

Result

The student satisfies Python and Machine Learning requirements but falls below the Data Analysis threshold.

Match Score = 66.67%

Technology Used

- Python
- CSV Dataset
- Rule-Based Matching Logic
